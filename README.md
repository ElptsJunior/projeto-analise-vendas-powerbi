# Análise de Vendas com SQL, Python e Power BI

## Objetivo
Este projeto tem como objetivo analisar dados de vendas e identificar padrões de faturamento e comportamento de mercado.

## Tecnologias utilizadas

- SQL
- Python
- Pandas
- Matplotlib
- Power BI

## Estrutura do projeto

dataset → dados de vendas  
python → scripts de análise  
sql → consultas analíticas  
dashboard → visualizações Power BI  

## Principais análises

- vendas por categoria
- vendas por região
- ticket médio
